Пример alert

```
basicModal.show({
    body: '<p>This is a dead-simple alert modal!<br>The message can be filled with anything you want.</p>',
    closable: true,
    buttons: {
        action: {
            title: 'Dismiss',
            fn: basicModal.close
        }
    }
})
```
