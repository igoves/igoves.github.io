**логическое выражение ? выражение 1 : выражение 2**  
  
**PHP**  

```
<?php//длинный кодif($a<b){$msg = 'a<b';}else{$msg = 'a>b';}//короткий аналог кода$msg = (a<b)? 'a<b': 'a>b';?>
```

  
  
**JS**  

```
if (document.all==true) then ie=1 else ie=0;сокращенное условие: var ie=document.all?1:0;
```
