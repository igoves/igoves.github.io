Эффекты: **pulse, jump, shake, twist, vanish, color**  

```
var rythm = new Rythm();
rythm.setMusic("path/to/sample.mp3");
rythm.start();
rythm.addRythm('pulse2', 'pulse', 0, 10, {
  min: 0.1,
  max: 1
})
```
