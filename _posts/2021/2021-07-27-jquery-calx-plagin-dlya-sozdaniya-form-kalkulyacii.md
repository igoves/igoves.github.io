![jQuery Calx - плагин для создания форм калькуляции](images/1431506870_untitled-4.png "jQuery Calx - плагин для создания форм калькуляции")

  
Мощный, но простой в использовании JQuery плагин для создания формы расчета или расчета таблицы, плагин разберет формулу и сделает расчет на его основе, при изменении формы результат обновится автоматически. Поддерживает разные форматы ввода, будто валюта или другие величины. Ваша формула может выглядеть например вот так:

```
($I*$P*((1 + $I)^$N)) / (1 - ((1 + $I)^$N))
```
