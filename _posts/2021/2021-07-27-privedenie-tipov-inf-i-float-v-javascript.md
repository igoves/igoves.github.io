**parseInt** - преобразует строковой аргумент в целое число нужной системы счисления. Если не использовать аргумент то используется 10-ая (аналог int в пхп)

```
parseInt("15px")
```

вернет 15  
  
**parseFloat** - преобразует строковой аргумент в число с плавающей точкой

```
parseFloat("3.14")
```

вернет 3.14
