categories: 
  - "js"
---

```
const timetable = require('timetable-fns')

// Simple difference in calendar dates
timetable.diff('2018-12-25', '2019-01-05')
// ==> 11

// Can also be negative
timetable.diff('2019-01-05', '2018-12-25')
// ==> -11
```
