![PreViewTube.js - jQuery плагин для анимации YouTube привьюшек](images/1433583282_untitled-1.png "PreViewTube.js - jQuery плагин для анимации YouTube привьюшек")

  
Поддерживает параметры Interval - через какой промежуток времени крутить картинки и Mode - вид анимации постоянная или при наведении.  
  
YouTube создает 4 thumbnails, 3 из них размерами 120px x 90px и один 480px ? 360px  
  
Пример использования

```
$('.someSelector').PreViewTube({
	'interval' : 500,
	'mode' : 'constant'
});
```
